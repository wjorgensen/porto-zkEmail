# `ServerActions.health`

Viem Action for retrieving the health status of the RPC server. Uses [`health`](/rpc-server/health) under the hood.

## Usage

:::code-group

```ts twoslash [example.ts]
import { ServerActions } from 'porto/viem'
import { client } from './config'

const health = await ServerActions.health(client) // [!code focus]
```

```ts twoslash [config.ts] filename="config.ts"
// [!include ~/snippets/viem/config.server.ts]
```

:::

## Parameters

This action takes no parameters.

## Return Value

See [`health`](/rpc-server/health#response).
