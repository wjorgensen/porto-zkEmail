# `ServerActions.getCapabilities`

Viem Action for retrieving the capabilities of the RPC server. Uses [`wallet_getCapabilities`](/rpc-server/wallet_getCapabilities) under the hood.

## Usage

:::code-group

```ts twoslash [example.ts]
import { ServerActions } from 'porto/viem'
import { client } from './config'

const capabilities = await ServerActions.getCapabilities(client) // [!code focus]
```

```ts twoslash [config.ts] filename="config.ts"
// [!include ~/snippets/viem/config.server.ts]
```

:::
